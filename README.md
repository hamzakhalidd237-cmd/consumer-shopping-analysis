# Consumer Shopping Trends — Exploratory Data Analysis

## Objective
Conducted comprehensive EDA on a consumer shopping trends dataset to uncover purchasing patterns across demographics, categories, seasons, and payment methods.

## Tools & Technologies
- Python (Pandas)
- Matplotlib & Seaborn for visualization
- Jupyter Notebook

## Business Questions Analyzed
1. What is the average consumer age across item categories?
2. Is there a significant difference in shopping behavior between genders?
3. Which item category generates the highest total purchase amount?
4. Do certain locations tend to spend more on shopping?
5. Is there a preference for particular sizes or colors within categories?
6. How does review rating correlate with purchase amount?
7. Does subscription status influence spending habits?
8. Which payment method is most commonly used?
9. Are there specific shipping types preferred by customers?
10. What is the impact of discounts on clothing sales across age groups?

## Methodology
1. Loaded and explored the shopping trends dataset
2. Used Pandas GroupBy for segmented analysis across demographics
3. Created cross-tabulations for gender vs size preferences
4. Built comparative bar charts for discount impact analysis
5. Analyzed correlations between review ratings and purchase frequency
6. Visualized trends using Matplotlib and Seaborn

## Key Findings
- Subscription status had minimal impact on average spending
- Discount usage patterns varied significantly across age groups
- Location-based spending showed clear regional preferences
